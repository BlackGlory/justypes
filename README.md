# justypes
Dumb types for TypeScript in the real world.

## The difference with [hotypes]

justypes are more like adjectives or nouns.

[hotypes]: https://www.npmjs.com/package/hotypes

## Install

```sh
npm install --save justypes
# or
yarn add justypes
```
